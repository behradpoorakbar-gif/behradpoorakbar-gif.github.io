What is a div element

The "div" element is like a bucket that groups sections of a webpage together.

what is one way you can identify or "target" a div element?

You target a div with the css code "class". This will target all other div's that have the same "class" css in their code.

Why would you use a div element?

You can repeat styling across a website using the same class without having to repeat the CSS.

What is border property?

The border property allows you to style the border around an element.

What are some of the different border properties?

  1. Borders style
  2. Border width
  3. Border color
  4. Border Sides
  5. Rounded borders

What is the diffrence between margins and padding?

A margin is the spacing around an element from everything around it while padding is the spacing content has from the element border.
